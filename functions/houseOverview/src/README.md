## Sample curl

```bash
curl --request GET \
  --url 'https://api.maxfrise.com/gethouses?landlord=email%2523audel91%2540gmail.com' \
  --header 'Content-Type: application/json'
```

## Sample event

```json
{
  "landlord": "email#audel91@gmail.com"
}
```
